Core Capability Modules

HackGPT wires together cutting edge components to form an integrated, end-to-end security assessment solution.

Advanced AI Engine

Supports OpenAI GPT-4 API and local offline models (Ollama/Llama) with automated fallback and token tracking.

Intelligent Reconnaissance

Automates footprinting, passive sub-domain collection, service identification, and active boundaries mapping.

Zero-Day Detection

Uses DBSCAN and Isolation Forest algorithms to group behavior logs and flag anomalies on the target host.

Safe Exploitation Engine

Validates findings safely without using shell=True, preventing unauthorized side-effects on test boundaries.

Enterprise Dashboard

Track scan progress, compliance maps, interactive findings lists, and audit trails from a single modern interface.

Compliance Mapping

Maps vulnerabilities directly to OWASP Top 10, NIST SP 800-53, ISO 27001, and SOC2 compliance matrices.

Interactive Terminal Sandbox

Experience the CLI commands of HackGPT directly in the browser.

HackGPT Shell Simulator
Type help to view available assessment commands.
